# HTML Health Checklist

執行 `python3 skills/html-health/check_html_health.py` 後確認：

- [ ] `<style>` / `</style>` 配對（數量相等）
- [ ] `<head>` / `</head>` 配對
- [ ] `<body>` / `</body>` 配對
- [ ] `<script>` / `</script>` 配對
- [ ] 解析後 `<body>` 有子元素（非空白）
- [ ] `<style>` 在 `<body>` 前關閉
- [ ] 關鍵區塊存在：nav / utilbar / drawer / 至少一個 table 或 section 或 mklab-*
- [ ] 7 個根目錄 HTML 全部健康
