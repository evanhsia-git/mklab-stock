🏗️ 整體架構概覽

層級技術選型說明
前端架構Web Components (Custom Elements v1)零建置、零 Node.js 依賴、原生瀏覽器支援、可直接 fork 運作
圖表引擎Lightweight Charts v4.1.3TradingView 官方輕量圖表庫，免費版支援 K 線、指標、畫線
模組系統ES Modules (import/export)瀏覽器原生模組，開發用相對路徑，部署配合
樣式系統CSS Custom Properties (Design Tokens)統一 var(--bg) var(--surface) 等，支援深/淺色主題即時切換
資料層data-client.js (單一來源)統一 stocks.json、industry.json、history/ 載入與快取
部署目標GitHub Pages (子路徑 /mklab-stock/)靜態檔案直出，無 Server 端渲染，CI 只跑 Python 抓資料
資料來源TWSE OpenAPI (免 Key) + yfinance (備援)收盤/PE/PB/殖利率每日抓；ROE/ROA 每週補；指數 K 線用 yfinance

2026-07-17
待修正Research 個股點擊未連動大圖
目前 loadKLine() 僅載入 window.TWII_KDATA（加權指數），點擊個股時個股摘要會同步更新，但大圖表仍顯示 TWII。 根因：無個股歷史 OHLC 序列資料源（stocks.json 僅含日收盤，無時序 K 線）。 若要支援，需在 fetch_data.py 增抓個股近 1 年日線（TWSE / FinMind / yfinance），寫入 history/{sym}.json， 並改 loadKLine() 依 currentSym 切換資料源。

待修正screener / index 漲跌幅資料校正

方案功能描述難度推薦

資料可行性確認（殖利率 / RS / EMA / MACD / IRR / Sharpe / 最大回撤）

指標可行性依賴
殖利率 (div)✅ 已有stocks.json 845 檔有值
IRR / XIRR✅ 可做（純前端數學）使用者輸入買賣現金流
RS / EMA / MACD / Sharpe / 最大回撤⚠️ 需歷史 OHLCfetch_data.py 需增抓個股近 1 年日線（目前僅指數有 twii_kdata.js，個股無時序）

mklab-stock/
├── index.html                    # 首頁：Market Health + 自選 + 評分 TOP10
├── mklab-stock-screener.html     # 多條件篩選器 (PE/PB/ROE/EPS/技術面)
├── mklab-stock-research.html     # 個股深度研究 (K線+MACD/KD+財報表)
├── mklab-stock-industry.html     # 33 產業動態 + 成分股明細
├── mklab-stock-watchlist.html    # 自選股管理 (localStorage 持久化)
├── mklab-stock-help.html         # 使用說明 + 資料來源 + 評分標準
├── mklab-stock-log.html          # 本頁：開發日誌與架構文件
├── assets/
│   ├── mklab-wc.js               # WC 元件庫 (4 大元件)
│   ├── mklab-core.js             # 共用核心 (Shell/Watch/Utils)
│   ├── data-client.js            # 統一資料存取層
│   └── lightweight-charts.min.js # 圖表庫 (vendor 複製)
├── data/
│   ├── stocks.json               # 全市場最新一日 (meta + stocks[])
│   ├── industry.json             # 33 產業聚合 + 區間績效
│   ├── history/YYYYMMDD.json     # 每日切片 (260 交易日)
│   ├── twii_kdata.js             # 加權指數 K 線 (window.TWII_KDATA)
│   ├── etf-shares.json           # ETF 發行張數 (估算 market_cap 用)
│   └── schema-version.json       # Schema 版本檢查
├── scripts/
│   ├── fetch_data.py             # GitHub Actions 每日抓取 (TWSE/yfinance)
│   ├── export_db.py              # 本機 DB 灌種 (一次性歷史)
│   ├── qa_gate.py                # 品質閘門 (部署前必過)
│   └── check_html_health.py      # HTML 結構健康檢查
├── .github/workflows/
│   ├── daily-update.yml          # 每日 17:00 抓收盤 + PE/PB/殖利率
│   ├── qa-gate.yml               # Push 觸發 QA Gate
│   └── html-health.yml           # HTML 變更觸發結構檢查
└── docs/mklab-stock-dav.md       # 架構決策記錄 (React vs WC 比較)
