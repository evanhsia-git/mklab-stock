v2.0 — Web Components 重構（2026-07-18）

  • 重構路線：放棄 React/Vite/npm（違反零依賴），改採 原生 Web Components / classic script。
  • 元件庫：assets/mklab-wc.js 註冊 <mklab-kline>、<mklab-datatable>、<mklab-drawer>、<mklab-router>。
  • 資料層：assets/js/data-client.js（IIFE、快取、ETag、內建鍵 stocks/indices/twiiKline），統一取代各頁散落 fetch()。
  • 表格元件：<mklab-datatable> 封裝 MKLAB.DataTable，支援 cols、page-size、default-sort、data-src、field-map、pager-id。
  • 所有主題頁：Market / Screener / Research / Industry / Watchlist / Help / Log 全部改為 WC 版本，移除所有 vanilla chart/table/drawer 內嵌邏輯，統一引入 assets/mklab-wc.js + assets/js/data-client.js。
  • Build 系統（已演進）：原 build/build_pages.py 將 templates/ + pages/ 組裝為根目錄完整 HTML；現已改為 build/template_sync.py（Template Synchronizer），根目錄 HTML 為唯一來源，pages/ 已移除。
  • QA Gate：Python/JSON/HTML/Link/Chart 全自動檢測，全綠 ALLOW DEPLOY。

v1.x — 靜態頁面 + vanilla JS（2026-07-15 前）

  • 純靜態 HTML + vanilla JS，無建置步驟。
  • 各頁面獨立 fetch('data/*.json')，程式碼重複、維護困難。
  • mklab-core.js 提供 MKLAB.DataTable、MKLAB.Drawer、MKLAB.Shell、MKLAB.Watch。
  • data/twii_kdata.js 掛載 window.TWII_KDATA 供 K 線圖讀取。
  • GitHub Actions 每日抓取 TWSE 收盤價、yfinance 補齊 ROE/ROA，產出 data/stocks.json、data/industry.json、data/history/*.json。

關鍵技術債清理

  • data/twii_kdata.js：原用 const TWII_KDATA → 不掛 window，改為 if(typeof window!=='undefined')window.TWII_KDATA=TWII_KDATA;。
  • Demo 頁路徑：prototypes/ 子目錄下 script src 改用絕對路徑 /vendor/...、/data/...、/assets/...，避免相對路徑 404。
  • 瀏覽器快取：Demo 頁 script src 加 ?v= cache-bust。
  • Custom Element 時序：呼叫元件方法（setData 等）必須等 customElements.whenDefined('mklab-kline') 後。
  • LightweightCharts v4 限制：無 series.getData()、無 overlay 繪圖工具（畫線/費波那契不生效）。
  • HTML 驗證：skills/html-health/check_html_health.py 的 escape_in_code_pre 必須真正轉義為 <style></style>（曾因自我替換 bug 導致誤報）。
  • QA Gate Links 檢查：data:/mailto:/javascript:/# 協議必須排除，否則 favicon 的 data:image/svg+xml 被誤判 404。
  • MKLAB 全域物件：mklab-core.js 用 IIFE + global.MKLAB = {...} 匯出，頁面 <script></script> 必須在 core.js 之後執行。
  • Build 後根目錄 HTML 需 git add：build/template_sync.py 同步輸出到專案根，CI 部署的是根目錄檔案，不是 pages/。
