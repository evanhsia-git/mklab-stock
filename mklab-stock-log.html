mklab-stock 開發日誌

📋 專案架構總覽

📄 前端頁面（5 大主題）

  • index.html — Market 首頁：全球股市、Market Health K線、情緒/原物料、市值前10、自選、TOP 10
  • mklab-stock-screener.html — 篩選器：策略篩選、自訂過濾
  • mklab-stock-research.html — 研究頁：個股選擇、K線圖、財報表
  • mklab-stock-industry.html — 產業動態：33 產業列表、成分股績效
  • mklab-stock-watchlist.html — 自選股：加入/排序/提醒/移除

⚙️ 核心模組 (assets/)

  • mklab-core.js — 全站唯一表格、抽屜、導覽、自選共用模組
  • data-client.js — 統一資料層：載入 stocks.json、industry.json
  • mklab-wc.js — Web Components 元件庫

📊 資料層 (data/)

  • stocks.json — 全市場股票資料（價格、PE/PB/ROE、產業等）
  • industry.json — 產業別績效（各區間報酬）
  • twii_kdata.js — TWII 加權指數 260 日 K 線
  • etf-shares.json — ETF 發行張數（market_cap 估算用）

🔧 後端/CI (scripts/)

  • fetch_data.py — 每日 TWSE OpenAPI 抓取資料
  • export_db.py — SQLite → JSON 匯出
  • qa_gate.py — 品質閘門：10 大類檢查
  • check_html_health.py — HTML 結構健康檢查

🌐 部署

  • GitHub Pages：main 分支自動部署
  • GitHub Actions：每日更新 + QA Gate + HTML Health
  • 子路徑部署：<base href="/mklab-stock/">

📦 Vendor

  • lightweight-charts.min.js — K 線圖引擎 v4.1.3
  • 零外部依賴（零 Node.js/npm 需求）

🔗 資料流

TWSE/TPEX OpenAPI → fetch_data.py → SQLite → export_db.py → data/stocks.json → 前端渲染

yfinance 備援（TWII K線、全球指數、ETF）

Build-Time 更新，靜態優先，無即時串流

📐 設計原則

  • 靜態優先：Build-Time 計算，Run-Time 只需靜態檔案
  • Fork First：Fork 後只需啟用 GitHub Pages
  • 零外部依賴：對終端使用者零 Node.js/npm
  • 手機優先：RWD 自適應，觸控友善
  • 資料誠實：示意資料標示清楚，真實資料來自 TWSE/yfinance

📅 開發歷程

  • v1 — 原型 HTML + Vanilla JS（5 主題頁面）
  • v2 — 共用模組化（mklab-core.js：表格/抽屜/導覽/自選）
  • v3 — 資料層強化（stocks.json、industry.json、QA Gate）
  • v4 — 功能完善（搜尋、排序、分頁、自選加入/移除）
