提醒（Alert）欄位說明

提醒欄顯示系統針對自選股設定的條件觸發狀態。當滿足以下任一條件時，該股提醒欄會標記警示：

  • 價格提醒：股價突破你設定的目標價或停損價
  • 技術提醒：RSI 進入超買（>80）或超賣（<20）區間
  • 法人提醒：三大法人連續買超 / 賣超達門檻
  • 異常提醒：單日漲跌超過 ±7% 或成交量異常放大

目前為示意階段：生產環境由 GitHub Actions 每日計算各股觸發狀態並寫入資料，屆時此欄會自動顯示對應警示標籤。

自選股中心（示意）。Alert 支援價格/技術/法人提醒，生產環境可由 Actions 產出。
